# Rusty-Polymorphia

A simple turn-based, self-playing adventure game.

# Dependencies

Crate rand

# Build instructions

1. Clone the repository
2. In terminal, navigate to Rusty-Polymorphia/src and run the following commands:
3. `cargo build`
4. `cargo run`
